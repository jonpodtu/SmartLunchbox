# Note to self
## GPIO Pins
Using the ESP32, we have the following connection for each of the different modules. All module use 5V. 

__MAX7219 LED Dot Matrix Module__
CLK: 2
SCL: 0
DIN: 4

__Photoresistor__
Analog Signal: 26

__18B20 Temperature Sensor__
Digital Signal: 27

__TTP223 Touch Sensor Module__
I/O: 15
